            RICE inputs RICE = (Reach × Impact × Confidence) ÷ Effort
            Reach value

            Reach description

            Impact (1–5)
            1 = Minimal, 5 = Massive. Effect per user. Used in RICE.
            Impact description
            Impact description
            How much this moves your core metric per user. Supports rich text formatting.
            Confidence (% 0–100)
            Percentage certainty. Used in RICE (normalized to 0–1 when > 1).
            Confidence description
            Confidence description
            Why you believe this will work; link data, experiments, or bets. Supports rich text formatting.
            Effort (1–5)
            1 = Tiny, 5 = Huge. Used as divisor in RICE formula.
            Effort description
            Effort description
            Rough sizing: people × weeks, dependencies, and complexity. Supports rich text formatting.

            Connect cloud storage

            Link this browser to your MongoDB workspace on Vercel. Data will be saved to the database instead of only localStorage.

            If your Vercel roadmap only has MONGODB_URI (no PM_API_SECRET), leave the key empty and click Connect — the app will sync automatically. Otherwise paste PM_API_SECRET, or use ?pm_api_key=YOUR_SECRET once in the URL.

            Use the same URL on every device: pm-prioritization-tool-six.vercel.app. Pull loads the latest MongoDB workspace; Save pushes this browser’s data. If counts differ, tap Pull from cloud.
